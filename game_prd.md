設計計一個幾 A 幾 B 的猜數字遊戲網頁：

1. 採用春天配色，扁平現代風格
2. 每個數字要有單獨的輸入框
3. 要具備 RWD
4. 版面配置如下

    ```
    +---------------------+
    | +--+ +--+ +--+ +--+ |
    | |  | |  | |  | |  | |
    | +--+ +--+ +--+ +--+ |
    |                     |
    | +-----------------+ |
    | |                 | |
    | |                 | |
    | |                 | |
    | |                 | |
    | |                 | |
    | |                 | |
    | |                 | |
    | |                 | |
    | +-----------------+ |
    +---------------------+
    ```
    - 上面是輸入框，下方是歷史區，每次猜測後，
        要在下方顯示幾 A 幾 B 的結果，A 和 B 要
        用不同色。
    - 在輸入框輸入單一數字後自動跳到下一個輸入框
5. 最多可猜 10 次
6. 猜中或到達 10 次沒猜中都要顯示訊息窗通知，
    關閉訊息窗之後重新開始遊戲
